# Beyond Binary Detector 🔍

A Python-based security prototype built during the hackathon to analyze transactional datasets, user behavior logs, and return patterns to detect potential retail misuse anomalies (such as "wardrobing" or serial return abuse). 

This core logic was prototyped in Google Colab to rapidly test and verify our detection methods before integration.

## 🚀 Key Features
* **Behavioral Anomaly Analysis:** Scans transaction profiles and return intervals to identify patterns indicative of wardrobing (e.g., rapid buy-and-return cycles for high-value items).
* **Misuse Flagging Engine:** Implements heuristic checks to dynamically score historical return ratios against normal buyer baseline distributions.
* **Rapid Prototyping:** Built and structured in Google Colab for quick testing and proof-of-concept validation.

## 🛠️ Built With
* **Python** - Core detection logic, data structures, and statistical scoring.
* **Google Colab** - Environment used for rapid prototyping, interactive data visualization, and execution.
