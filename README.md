# Beyond Binary Detector 🔍

A Python-based security prototype built during the hackathon to analyze file structures and detect potential binary anomalies. 

This core logic was prototyped in Google Colab to rapidly test and verify our detection methods before integration.

## 🚀 Key Features
* **Binary File Analysis:** Scans file structures to identify potential security flags.
* **Rapid Prototyping:** Built and structured in Google Colab for quick testing and proof-of-concept validation.

## 🛠️ Built With
* **Python** - Core detection logic and scripting.
* **Google Colab** - Environment used for rapid prototyping and execution.
